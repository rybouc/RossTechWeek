## Data Dictionary - E-commerce Dataset

This document provides detailed descriptions of all columns in the three main datasets: events.csv, items.csv, and users.csv.

### Dataset Overview

| Dataset | Description |
|---------|-------------|
| events.csv | User interaction events (purchases, cart additions, checkouts) |
| items.csv | Product catalog with pricing and categorization |
| users.csv | User profiles with lifetime value and registration data |

---

### events.csv - User Event Data

| Column Name | Data Type | Description | Example Values | Notes |
|-------------|-----------|-------------|----------------|-------|
| `user_id` | Integer | Unique identifier for each user | 2133, 5789, 5808 | Foreign key to users.id |
| `ga_session_id` | Integer | Web cookie session identifier | 16909, 16908, 4267 | Groups events within a user session |
| `country` | String | Two-letter country code where event occurred | US, SE, CA, GB | ISO country codes |
| `device` | String | Device type used for the event | mobile, desktop, tablet | Categories: mobile, desktop, tablet |
| `type` | String | Type of user interaction | purchase, add_to_cart, begin_checkout | Event funnel stages |
| `item_id` | Integer | Unique identifier for the product | 94, 425, 1, 62 | Foreign key to items.id |
| `date` | DateTime | Timestamp when the event occurred | 2020-11-01 00:27:14 | UTC timezone, YYYY-MM-DD HH:MM:SS format |

Event Types
- **add_to_cart**: Items added to shopping cart
- **begin_checkout**: Checkout process initiated
- **purchase**: Completed transactions

---

## items.csv - Product Catalog

| Column Name | Data Type | Description | Example Values | Notes |
|-------------|-----------|-------------|----------------|-------|
| `id` | Integer | Unique product identifier | 0, 1, 2, 3 | Primary key, referenced by events.item_id |
| `name` | String | Product name/title | "Land & Sea Cotton Cap", "KeepCup" | Full product description |
| `variant` | String | Product variant/size/color | "Single Option Only", " LG", " MD", " SM" | Size, color, or style variations |
| `category` | String | Product category classification | Apparel, Drinkware, Campus Collection | Business category groupings |
| `price_in_usd` | Float | Product price in US dollars | 14, 28, 20, 22 | Retail price, positive values |

Product Categories
- **Apparel**: Clothing items (t-shirts, hoodies, etc.)
- **Drinkware**: Bottles, cups, mugs
- **Campus Collection**: Location-specific merchandise
- **Clearance**: Discounted items
- **New**: Recently launched products
- **Shop by Brand**: Brand-specific collections
- **Uncategorized Items**: Products not yet categorized

---

### users.csv - Customer Profiles

| Column Name | Data Type | Description | Example Values | Notes |
|-------------|-----------|-------------|----------------|-------|
| `id` | Integer | Unique user identifier | 0, 1, 2, 3 | Primary key, referenced by events1.user_id |
| `ltv` | Integer | Customer lifetime value in USD | 0, 231, 102, 75 | Total revenue generated by customer |
| `date` | DateTime | User registration/first seen date | 2020-10-13 05:08:47 | Account creation timestamp |
